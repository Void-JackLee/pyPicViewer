# PicViewer

A Picture Viewer that allows photographers to conveniently compare and select pictures.

# Install

* (optinal)
```
conda create --name picv python=3.12
conda activate picv
```

* install with pip
```
pip3 install -r requirements.txt
```

# Usage

```
# start application directly
python main.py

# strat with directory or image
python main.py [image_path | dir_path]
```
